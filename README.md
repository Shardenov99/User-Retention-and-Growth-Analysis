## 1. Monthly Cohort Retention Rate
 -  Description: Tracks how user retention changes month-to-month across cohorts.
 -  Key Insights: Retention averages 39.25% in Month 0 and drops below 25% by Month 3â€“4, showing weak repeat engagement and declining early-stage conversion.
   
<br>

## 2. Conversion Funnel by Traffic Source
  -  Description: Analyzes purchase funnel stages for mobile vs. web users.
  -  Key Insights: Mobile drives ~90% of traffic but converts at the same rate as web. Major drop-off occurs after Add to Cart, suggesting checkout issues or customer hesitation.

<br>

## 3. RFM Customer Segmentation
 - Description: Segments customers by recency, frequency, and monetary value.
 - Key Insights: Champions and Big Spenders (~20%) drive most revenue, while Occasional and Lost users (~54%) dominate the user base, highlighting weak customer retention.
<br>

## 4.  Summary
- **Revenue growth is slowing** because the customer base is shifting toward low-value, infrequent buyers.  
- **New users struggle to become loyal, high-value customers**, limiting long-term revenue potential.  
- **Loyal, profitable customers are a minority**, and many previously active users have lapsed.
